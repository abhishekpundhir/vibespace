## How To Setup Project
- Clone Project in Terminal (Code Editore) 
   ```
  git clone "https://github.com/abhishekpundhir/vibespace.git"
   ```
- Navigate To Project 
   ```
  cd vibespace
   ```
- Install Project Dependencies
   ```
  npm i
   ```
- Run Project
   ```
  npm run dev
   ```
